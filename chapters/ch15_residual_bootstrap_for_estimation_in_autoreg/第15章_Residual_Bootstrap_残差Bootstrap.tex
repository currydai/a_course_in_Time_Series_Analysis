\section*{本章定位}

本章对应原文 \textit{Residual Bootstrap for estimation in autoregressive processes}。首版讲义先按照原文的章节主线整理核心概念、教学解释和 Python 工程实践；后续精修时，可继续把原 PDF 中的证明、练习和细节推导逐节扩展进来。

本章范围：残差 Bootstrap 与 AR 参数估计抽样性质。

\section{核心知识点}

\begin{itemize}
  \item 残差 Bootstrap 用拟合模型的残差近似创新分布。
  \item 重抽样后重新生成序列并重新估计参数，可以近似估计量分布。
  \item 该方法依赖模型拟合质量和残差近似独立性。
\end{itemize}
